**security**

> ビジネスロジックの欠陥（認可バイパス・処理順序の攻撃）は標準的なSASTやDASTでは検出できない。AIでAPIコールグラフを生成し脆弱性を特定する新しいアプローチ。

Cloudflareが公開したWeb and API Vulnerability Scannerは、AIがAPIコールグラフを構築することで、従来の防御ツールでは見つけられなかった論理的欠陥を発見する。ステートフル（状態を持つ）なAPI呼び出しのシーケンスを分析することで、認証済みエンドポイントの認可バイパスや、複数ステップにまたがる処理の脆弱性を検出する。

REST APIを中心に、従来のスキャナが「個別リクエスト」しか見られなかった限界を、AIによるフロー解析で補う設計になっている。
